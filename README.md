🚀 SkillNest – Full Stack Course Platform

SkillNest is a full-stack MERN application that allows users to discover, enroll in, and manage courses, while providing admins with full control over course content through CRUD operations.

🌐 Live Links
🔗 Frontend: https://skillnest-fullstack-course-platform.vercel.app

🔗 Backend: https://skillnest-backend-kvc2.onrender.com

🔗 Documentation: https://skillnestproject.wordpress.com


✨ Features
👤 User
User authentication (JWT-based login/signup)
Browse all available courses
Enroll in courses
Personalized course recommendations
Responsive UI with light/dark mode


🛠 Admin
Secure admin login
Add new courses
Edit existing courses
Delete courses
Real-time course dashboard management


🧱 Tech Stack
Frontend: React.js, CSS, Axios, React Router
Backend: Node.js, Express.js
Database: MongoDB Atlas
Authentication: JWT (JSON Web Tokens)
Deployment: Vercel (Frontend), Render (Backend)

⚙️ How To Run Locally-
Backend
cd Skillnest-backend
npm install
npm run dev

Frontend
cd client
npm install
npm start


🧠 Key Concepts
RESTful API design
JWT-based authentication & authorization
Role-based access (Admin vs User)
Protected routes
Environment-based API configuration
Full-stack deployment (Vercel + Render)
